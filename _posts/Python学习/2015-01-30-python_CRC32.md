## **python CRC32方法**

```python
	def _crc32(self, v): 
    """ 
    Generates the crc32 hash of the v. 
    @return: str, the str value for the crc32 of the v 
    """ 
    return '0x%x' % (binascii.crc32(v) & 0xffffffff)
```
