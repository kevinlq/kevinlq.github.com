### 忽略文件修改

- 忽略规则

```
*.o
/Manager　　 # 仅仅忽略项目根目录下的 Manager　文件，不包括 subdir/Manager　
build/ 　　 # 忽略 build/ 目录下的所有文件
doc/*.txt 　　# 会忽略 doc/notes.txt 但不包括 doc/server/arch.txt

```

>当重新修改忽略文件后，规则不能立马生效，需要先清除缓存，在重新 `add`

```
git rm -r --cached .
git add .
git commit -m 'update .gitignore'
```
